# survey-bot

A speech-enabled Chat-bot.

Get the user to answer questions and talk about themselves!

Uses:

* [Node.js][]
* [NLP.js][]
* [Webchat.js][]
* [MS Speech SDK][speech-sdk]

## Usage

```
npm i
npm start
npm test
```
